import numpy as np

def fun1(n):
    sum = 0
    for i in range(n):
        sum += 1
    return sum


def runtime(f, n):
    #at the start of cell, define <tic> variable
    tic = time.time()
    # Runs the function (Execute)
    result = f(n)
    #at the end of cell, define <toc> variable
    toc = time.time()
    #store execution time in variable. Note the unit of the result is seconds
    time_taken = toc - tic 
    return time_taken

# Create list of x and y
x_list = [1000000, 2000000, 3000000, 4000000, 5000000, 6000000]
y_list = [runtime(fun1, n) for n in x_list]

plt.figure(figsize = (10,6))

plt.title('runtime of fun1 as a function of n')
plt.xlabel('n')
plt.ylabel('runtime')
plt.grid(True, which="both", linestyle="--", linewidth=0.5)
plt.show()

3)

  1. Show that \((n + 1)^5\) is \(O(n^5)\).

  2. Show that \(2^{n+1}\) is \(O(2^n)\).

  3. Show that \(n\) is \(O(n \log n)\).

Solution:

  1. Since \(\lim_{n \to \infty} \frac{(n+1)^5}{n^5}\) = \(\lim_{n \to \infty} (\frac{n+1}{n})^5\) = \(\lim_{n \to \infty} (1 + \frac{1}{n})^5\) = \(1\).

Because \(lim_{n \to \infty}\frac{f(n)}{g(n)}\) is constant, (n+1)^5 grows at the same rate as n^5.

Therefore, \(n^5\) is the theta of \((n+1)^5\) and then also the Big-O of \((n+1)^5\).

  1. Since \(\lim_{n \to \infty} \frac{2^{n+1}}{2^n}\) = \(\lim_{n \to \infty} 2\) = \(2\)

Because \(lim_{n \to \infty}\frac{f(n)}{g(n)}\) is constant, 2^{n+1} grows at the same rate as \(2^n\).

Therefore, \(2^n\) is the theta of \(2^{n+1}\) and then also the Big-O of \(2^{n+1}\).

5) Order the following 14 functions by growth rate.

\(n, \sqrt n, n^{1.5}, n^2, n\log n, n \log \log n, n \log^2 n, n \log (n^2), 2/n, 2^n, 2^{n/2}, 37, n^2 \log n, n^3\)

Solution:

We will classify the order of growth of each function first.

\(n\) is linear.

\(\sqrt n\) has lower growth rate compared to \(n\) since \(\lim_{n \to \infty}\frac{\sqrt n}{n} = \lim_{n \to \infty}\frac{1}{\sqrt n} = 0\).

\(n^{1.5}\) has higher growth rate compared to \(n\) since \(\lim_{n \to \infty}\frac{n^{1.5}}{n} = \lim_{n \to \infty}\sqrt n\) which goes to \(\infty\).

\(n^2\) has higher growth rate compared to \(n^{1.5}\) since \(\lim_{n \to \infty}\frac{n^{2}}{n^{1.5}} = \lim_{n \to \infty}\sqrt n\) which goes to \(\infty\).

So now we have \(\sqrt n, n, n^{1.5}, n^2\) in increasing order of growth.

\(n\log n\) has higher growth rate compared to \(n\) since \(\lim_{n \to \infty}\frac{n \log n}{n} = \lim_{n \to \infty}\log n\) which goes to \(\infty\).

\(n\log n\) has lower growth rate compared to \(n^{1.5}\) since \(\lim_{n \to \infty}\frac{n \log n}{n^{1.5}} = \lim_{n \to \infty}\frac{\log n}{\sqrt n} = 0\) since \(\sqrt n\) grows much faster than \(\log n\) when n is large(Imagine \(2^{\sqrt n}\) which grows exponentially and \(2^{\log n} = n\) which grows linearly).

\(n\log \log n\) has higher growth rate compared to \(n\) since \(\lim_{n \to \infty}\frac{n \log \log n}{n} = \lim_{n \to \infty}\log \log n\) which goes to \(\infty\).

\(n\log \log n\) has lower growth rate compared to \(n \log n\) since \(\lim_{n \to \infty}\frac{n \log \log n}{n \log n} = \lim_{n \to \infty}\frac{\log \log n}{\log n} = 0\) since \(\log n\) grows faster than \(\log \log n\) (Imagine \(2^{\log \log n} = \log n\) which grows logarithmically and \(2^{\log n} = n\) which grows linearly, Logarithmically is slower than linear).

So now we have \(\sqrt n, n, n \log \log n, n \log n, n^{1.5}, n^2\) in increasing order of growth.

\(n \log^2 n\) has higher growth than \(n \log n\) since \(\lim_{n \to \infty}\frac{n \log^2 n}{n \log n} = \lim_{n \to \infty}\log n\) which goes to \(\infty\).

\(n \log^2 n\) has lower growth than \(n^{1.5}\) since \(\lim_{n \to \infty}\frac{n \log^2 n}{n^{1.5}} = \lim_{n \to \infty}\frac{\log^2 n}{\sqrt n} = 0\) (Imagine writen \(n = 2^k\) for some real number \(k\), we have \(\log^2 n = k^2\) which is quadratic while \(\sqrt n = n^{k/2}\) which is exponential).

\(n \log (n^2) = 2n \log\) has the same growth rate as \(n \log n\) since \(\lim_{n \to \infty}\frac{2n \log n}{n \log n} = \lim_{n \to \infty}2 = 2\) which is a constant.

So now we have \(\sqrt n, n, n \log \log n, n \log n, n \log n^2, n \log^2 n, n^{1.5}, n^2\) in increasing order of growth.

\(37\) is a constant, so it has lower growth than \(\sqrt n\).

\(2/n\) is decreasing when n increases, so it has lower growth than \(37\).

So now we have \(2/n, 37, \sqrt n, n, n \log \log n, n \log n, n \log n^2, n \log^2 n, n^{1.5}, n^2\) in increasing order of growth.

\(n^2 \log n\) has higher growth rate than \(n^2\) since \(\lim_{n \to \infty}\frac{n^2 \log n}{n^2} = \lim_{n \to \infty}\log n\) which goes to \(\infty\).

\(n^3\) has higher growth rate than \(n^2 \log n\) since \(\lim_{n \to \infty}\frac{n^3}{n^2 \log n} = \lim_{n \to \infty}\frac{n}{\log n}\) which goes to \(\infty\) (\(n\) grows faster than \(\log n\)).

\(2^{n/2}\) grows exponentially, so it grows faster than \(n^3\).

\(2^n\) grows faster than \(2^{n/2}\) since \(\lim_{n \to \infty}\frac{2^n}{2^{n/2}} = \lim_{n \to \infty}2^{n/2}\) which goes to \(\infty\).

Therefore, the order of function by increasing growth rate is \(2/n, 37, \sqrt n, n, n \log \log n, n \log n, n \log n^2, n \log^2 n, n^{1.5}, n^2, n^2 \log n, n^3, 2^{n/2}, 2^n\).
